# Política de Segurança

## ⚠️ Aviso Importante

Este repositório contém código **intencionalmente vulnerável** para fins educacionais.

**NÃO reporte as vulnerabilidades deste projeto** — elas foram implementadas propositalmente como material de estudo do OWASP Mobile Top 10.

## Uso responsável

Este projeto destina-se exclusivamente a:

- Estudo de segurança mobile em ambientes controlados
- Aprendizado prático das vulnerabilidades do OWASP Mobile Top 10
- Treinamento de equipes de segurança
- Fins acadêmicos e de pesquisa

## O que NÃO fazer

- Não utilize este código em aplicações de produção
- Não utilize as técnicas demonstradas contra sistemas sem autorização
- Não distribua APKs deste projeto para dispositivos reais
